# Welcome to Lyyti Design System

The Lyyti Design System is here to help you build an even more lovable Lyyti.
The whole of the Design System can be found in [**Zeroheight**](https://lyyti.zeroheight.com/styleguide/s/24777/p/593925-lyyti-design-system).

This version of Design System in Storybook is meant mostly for developers and others interested in Lyytis UI components in isolation from any app's business logic, data, and context.

We mainly use ready-made components from libraries like [**MUI**](https://mui.com/) and add styles to them to match Lyytis design. Changes to component functionality are done based on needs and kept quite minimal.
